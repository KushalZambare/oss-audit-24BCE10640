### Step 2: Give execution permission

```bash

```

### Step 3: Run each script

#### ▶️ Script 1: System Identity Report

```bash
./1_system_identity.sh
```

✔ Displays system information like kernel, user, uptime, and date.

---

#### ▶️ Script 2: Package Inspector

```bash
./2_package_inspector.sh
```

✔ Checks if Python is installed and shows version info.

---

#### ▶️ Script 3: Disk and Permission Auditor

```bash
./3_disk_auditor.sh
```

✔ Displays directory size, permissions, and ownership.

---

#### ▶️ Script 4: Log File Analyzer

```bash
./4_log_analyzer.sh
```

✔ Counts occurrences of a keyword (e.g., "error") in a log file.

---

#### ▶️ Script 5: Open Source Manifesto Generator

```bash
./5_manifesto_generator.sh
```

✔ Generates a personalized open-source philosophy statement.
